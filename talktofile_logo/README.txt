TalkToFile — Logo Asset Pack
Mark: the speech-bubble option (folded file + reply bubble). Accent: terracotta #C2410C. Ink: #0e1413.

FOLDERS
  svg/   scalable vector — use these wherever possible (web, print, app stores)
  png/   raster fallbacks — use where SVG isn't accepted (some social uploads, email)

SVG FILES
  mark-color.svg      Primary mark. Dark file + terracotta bubble. For light backgrounds.
  mark-black.svg      Single-ink black. Stamps, faxes, one-color print, watermarks.
  mark-white.svg      All-white reversed mark. For dark/photo backgrounds.
  mark-white-accent.svg  Reversed white mark with terracotta bubble. For dark backgrounds
                      where you want the accent to pop (transparent).
  app-icon.svg        Solid terracotta rounded tile, white mark. App icon + social avatar.
  app-icon-dark.svg   Dark rounded tile variant for light/grid avatar contexts.
  favicon.svg         Simplified mark tuned to stay legible at 16px.
  lockup-color.svg    Horizontal mark + "Talktofile" wordmark (Merriweather italic). For light bg.
  lockup-black.svg    Mono black lockup.
  lockup-white.svg    Reversed lockup for dark bg.
  (lockup SVGs reference the Merriweather webfont; convert text to outlines in your
   vector tool if you need them fully self-contained.)

PNG FILES
  mark-color / mark-black / mark-white   1024px & 512px, transparent.
  mark-white-accent                      1024px & 512px, transparent.
  app-icon-1024 / -512, app-icon-dark-1024   square, full-bleed.
  favicon-16 / -32 / -48 / -64           browser tabs.
  favicon-180                            Apple touch icon.
  lockup-color / lockup-black            on white.
  lockup-white                           on dark.
  lockup-color/black/white-transparent   mark + wordmark, NO background (transparent).

NOTE ON TRANSPARENCY
  All SVGs are transparent by nature (no background) except app-icon* and favicon,
  which intentionally carry their colored tile.
  The mark PNGs (mark-color/black/white) are already transparent.
  For the lockup as a transparent PNG, use the *-transparent.png files above.

QUICK PICKS
  Website favicon ............ png/favicon-32.png (or svg/favicon.svg)
  App store / iOS icon ....... svg/app-icon.svg or png/app-icon-1024.png
  Social profile picture ..... png/app-icon-1024.png  (square, fills the circle crop)
  Email signature / nav ...... svg/lockup-color.svg or png/lockup-color-transparent.png
  Dark-mode header ........... svg/lockup-white.svg
  Merch / one-color print .... svg/mark-black.svg

Minimum size: keep the mark at 24px or larger; below that use favicon.svg.
Clear space: leave at least the height of the bubble around the mark on all sides.
